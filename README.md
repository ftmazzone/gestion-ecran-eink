# Installation

Voir [canvas](https://github.com/Automattic/node-canvas).

# Exemple 

- Allumer l'écran
- Afficher un texte
- Allumer les diodes éclairant l'écran
- Détecter les mouvements devant l'écran

```bash
cargo run --example afficher_jour
```
